# Eventonow — Frontend

Este es el frontend de Eventonow (Next.js 16 / React 19). La documentación del proyecto vive en [`../docs/`](../docs/) — empezá por [`../CLAUDE.md`](../CLAUDE.md) y [`../docs/PROJECT.md`](../docs/PROJECT.md).

## Arrancar en desarrollo

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000). Requiere el backend corriendo (`eventonow-back/`) y `NEXT_PUBLIC_API_URL` apuntando a él — ver [`../docs/DEPLOYMENT.md`](../docs/DEPLOYMENT.md).

Antes de escribir código, leer `AGENTS.md` (Next.js 16 tiene breaking changes respecto a versiones anteriores) y [`../docs/FRONTEND.md`](../docs/FRONTEND.md).
